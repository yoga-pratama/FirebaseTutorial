# FirebaseTutorial IOS Swift

Tutorial Auth meggunakan firebase di Swift IOS

Sample Code login yang terhubung ke firebase .

Requirement 
- Akun Firebase
- Framework Cocoapods

Tambah line berikut di podfile
 - pod 'Firebase/Core'
 - pod 'Firebase/Auth'
 - pod 'GoogleSignIn' .


Pada APPDelegate dan view Controller  tambahkan 

import Firebase
import GoogleSignIn
